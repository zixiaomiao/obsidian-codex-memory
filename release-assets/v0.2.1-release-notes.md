# Codian v0.2.1

> 旧项目名称: obsidian-codex-memory

This release adds double-click installers for macOS and Windows.

## Assets

- `install-macos.command`
- `install-windows.cmd`

## Usage

Download the installer for your operating system and double-click it.

The installer will:

- clone or update `zixiaomiao/codian`
- install it into the user's plugin directory
- register it in the Codex personal marketplace

After installation, open Codex and enable **Codian**.
