# Codian v1.0.1

## 中文说明

这个版本修复 Obsidian vault 写入位置。

- 第一次运行 `init` 时，会在 Obsidian vault 根目录创建 `codian/`。
- 后续所有 Codian 记忆文件都会写入 `codian/` 目录内。
- 默认路径现在是：
  - `codian/README.md`
  - `codian/AGENTS.md`
  - `codian/10-Context-上下文/project-summary.md`
  - `codian/20-Memory-记忆/`
  - `codian/30-Logs-日志/codex-session-summary.md`
  - `codian/40-Workflows-工作流/codex-memory-workflow.md`
  - `codian/90-Archive-归档/`
- 如果已通过 `--memory-rel`、`CODIA_MEMORY_REL` 或配置文件自定义路径，仍然尊重自定义路径。

## English

This release fixes where Codian writes inside an Obsidian vault.

- First `init` creates `codian/` at the Obsidian vault root.
- All default Codian memory files are written under `codian/`.
- Explicit custom paths from `--memory-rel`, `CODIA_MEMORY_REL`, or config are still respected.
