# Codian v1.0.0

This release promotes the plugin to Codian v1.0.0 and syncs the expanded Codex skill instructions into the GitHub plugin repository.

## 中文说明

这个版本把 Obsidian Codex Memory 正式升级为 Codian v1.0.0，并把当前正在使用的完整 Codex skill 说明同步到插件仓库。

主要变化：

- 插件身份统一为 `codian`。
- `.codex-plugin/plugin.json` 版本更新为 `1.0.0`。
- bundled skill 使用仓库根目录的 `SKILL.md`。
- 安装脚本会把完整仓库同步到 `~/.codex/skills/codin GitHub`，再把必要的运行文件同步到 `~/.codex/skills/codin`，让其他电脑安装后也能直接获得独立 skill。
- skill 说明补全了 Obsidian 长期记忆的完整工作流：
  - 精简读取记忆
  - 按关键词匹配历史日志
  - 写入前确认的安全追加流程
  - 项目摘要生成
  - 分类记忆生成
  - vault 路径自动发现规则
- 记忆脚本继续使用新的结构化 Obsidian vault 布局：
  - `README.md`
  - `AGENTS.md`
  - `10-Context-上下文/`
  - `20-Memory-记忆/`
  - `30-Logs-日志/`
  - `40-Workflows-工作流/`
  - `90-Archive-归档/`

## Highlights

- Renames the Codex plugin identity to `codian`.
- Updates the plugin manifest version to `1.0.0`.
- Moves the bundled skill to the repository root as `SKILL.md`.
- Installs the repository contents into `~/.codex/skills/codin GitHub` and the runtime files into `~/.codex/skills/codin` so other machines get the standalone skill too.
- Expands the skill instructions with the full Obsidian memory workflow:
  - compact memory reads
  - keyword-matched historical log retrieval
  - safe append workflow
  - project summary generation
  - categorized memory generation
  - vault discovery rules
- Keeps the memory script focused on the structured Obsidian vault layout:
  - `README.md`
  - `AGENTS.md`
  - `10-Context-上下文/`
  - `20-Memory-记忆/`
  - `30-Logs-日志/`
  - `40-Workflows-工作流/`
  - `90-Archive-归档/`

## Install

macOS or Linux:

```bash
curl -fsSL https://raw.githubusercontent.com/zixiaomiao/codian/main/install.sh | bash
```

Windows PowerShell:

```powershell
irm https://raw.githubusercontent.com/zixiaomiao/codian/main/install.ps1 | iex
```
