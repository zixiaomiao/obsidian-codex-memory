# Codian v1.0.2

This release aligns the local Codian runtime and the GitHub mirror, and standardizes the Obsidian memory root to `Codian Memory/`.

## 中文说明

这个版本主要做了两件事：

- 以本机 `~/.codex/skills/Codian` 为准，把正在使用的 `SKILL.md` 和 `scripts/obsidian_memory.py` 同步回 `Codian GitHub`。
- 把 Obsidian 长期记忆目录正式统一为 `Codian Memory/`，避免继续写入旧的 `codian/` 路径。

结果是：

- 本地运行目录和 GitHub 镜像目录的核心脚本保持一致。
- `Codian Memory/` 成为默认记忆根目录。
- 后续写入、项目摘要和分类归纳都会落到新的目录结构。

## Highlights

- Syncs `SKILL.md` and `scripts/obsidian_memory.py` from the live local Codian runtime back into the GitHub mirror.
- Standardizes the Obsidian memory root as `Codian Memory/`.
- Keeps the local `Codian` folder and the `Codian GitHub` folder aligned for the core runtime files.
